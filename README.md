# app-ov7670
ov7670 camera module driver and software  

* 基本的には参考文献の追試であるが，
linux+pythonで画像表示したいため，画像表示プログラムはpythonバージョンで作成する．  
* ステップを踏んで開発するため，バージョンごとの目標を記載する．  
* 余裕あれば，nanoバージョンも作成する．
